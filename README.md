# poc-rabbitmq
Poc vagrant + rabbitmq


<h6>Inicialiar os ambientes com vagrant - node1 e node2 </h6>
```
  cd poc-rabbitmq/
  vagrant up
```  
  
<h6>Verificar o status das maquinas(node1/node2)</h6>
```
  vagrant status
  
node1                     running (virtualbox)
node2                     running (virtualbox)
```

<h6>Acesso as maquinas</h6>
```
  vagrant ssh node1
  vagrant ssh node2
```
